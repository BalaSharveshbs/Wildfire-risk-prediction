About the Wildfire Risk Prediction Model

This model predicts wildfire risk based on temperature, humidity, windspeed, rainfall, location, and vegetation type.

  • Model: Random Forest Classifier
  • Input Format: Case-insensitive for location and vegetation
  • Output: Risk Level (Low, Medium, High)
  • Tech Stack: Flask, scikit-learn, HTML, CSS
